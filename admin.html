Troubleshooting

Tests aren't running on an instance

Check the table sn_atf_tg_sn_boq on the instance — it shows tests currently running or queued.

If nothing's there, the test never started; if it's stuck, the test agent may be hung.

Repair the ATF plugin on an instance
  1. Hop to the attendee's instance (your admin session works across all workshop instances).
  2. From the navigator, type v_plugin.list and press Enter.
  3. Search for ID = com.glide.automated_testing_impl.sn_atf_tg.hosted.
  4. Open the form.
  5. At the bottom of the form, click Activate / Repair.
Help an attendee re-claim

Find their instance number → use Free a specific instance below to clear it → tell them to refresh the workshop URL and tap Claim my instance again.

The freed slot goes to the back of the queue, so they may get a different number this time.
